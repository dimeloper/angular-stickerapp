#AngularJS Sticker Application

Bower Dependencies:
- angular
- angular-mocks
- angular-resource
- angular-animate
- angular-ui-router
- angular-bootstrap
- bootstrap
- angular-file-upload
- angular-dragdrop

Npm and bower needed to be installed.

Run "npm start" to install needed dependencies and start a local server.
